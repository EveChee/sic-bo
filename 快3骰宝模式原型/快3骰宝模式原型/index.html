玩法说明：大小：大：开奖骰子三个号的和值为11~18即中奖；小：开奖骰子三个号的和值为3~10即中奖

                 单双：单：开奖骰子三个号的和值为单数即中奖；双：开奖骰子三个号的和值为双数即中奖

玩法说明：可以选择一个或多个和值进行投注，所选和值与开奖骰子三个号的和值相同即中奖

玩法说明：单选：可以选择一个或多个豹子号进行投注，开奖骰子为所选豹子号即中奖

                 通选：对所有豹子进行投注，开奖骰子为豹子即中奖

玩法说明：可以对一个或多个二同号进行投注，开奖骰子包含所选二同号即中奖

玩法说明：可以对一个或多个二不同号进行投注，开奖骰子包含所选二不同号即中奖

玩法说明：从六个号码中选择一个或多个号码进行投注，若开奖骰子包含所选号码即中奖

玩法说明：从三连号中选择一组或多组进行投注，若开奖骰子包含所选三连号即中奖

筹码种类包括：1元、5元、10元、20元、50元、100元、1K元、5K元共八种

确认投注：在桌面押注后，点确认投注，投注生效

撤销投注：在桌面押注后，点撤销投注，投注注码取消

翻倍投注：当前桌面的所有投注筹码乘以2  若余额不足以翻倍投注，提示“余额不足”
